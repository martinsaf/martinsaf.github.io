>_ home-soc/

🔧 active

>_ overview

Home laboratory environment for hands-on practice with IT systems, monitoring, and security detections. A sandbox to learn and experiment with Windows & Linux administration, network configuration, automation, and security telemetry.

>_ network architecture

    [Windows Laptop] 192.168.200.1
            ├── (Ethernet) ── [Ubuntu Server] 192.168.200.2  (Wazuh)
            ├── (Bridge)   ── [Windows Server VM] 192.168.200.10 ()
            ├── (Bridge)   ── [Windows 10 VM]  192.168.200.20 (Sysmon)
            └── (Bridge)   ── [Ubuntu VM]      192.168.200.30 (auditd)
